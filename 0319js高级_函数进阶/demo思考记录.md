#### 递归
##### 1. 利用递归求数学题 求1*2*3*...*n 阶乘
1. 结束条件 n==1 return 1；
2. 单层循环 n * f(n-1)

##### 2. 利用递归求斐波那契数列（前两项相加 == 第三项）
用户输入一个数字 n 就可以求出 这个数字对应的序列值  n 从 1 开始算
1. 单层循环 f(n) = f(n-1) + f(n-2)  return f(n-1)+f(n-2)
2. 退出循环 f(n-2) == 1; f(n-1) == 1;  所以是 n == 2 || n == 1 -- > return 1;
